# Communication Ports
  These are the ports mapping the sensors in the CPU:
  - Sick laser: recognized automatically
  - GPS: /dev/gps
  - IMU: /dev/imu
  - Metal detector: recognized automatically
  - Robot base: /dev/rosaria

# Power Distribution

 from robot:
 - Sick laser: 12v
 - GPS:  5v
 - IMU:  5v
 - Metal detector: 12v
 - Robot base: 12v
 - USB hub: 5v
 - wifi: 5v

 from own battery(Power 4200mah 12v):
 - Raspberry PI: 5v
