# HRATC 2017 Launch Files

Launch files for the HRATC 2017 robot

## Files
###trouble.launch
Full launch file. Runs the following packages (dependencies):
* RosAria
* um6
* nmea_navsat_driver
* sick_tim
* robot_localization
* tf
* metal_detector_msgs
* hokuyo_node
